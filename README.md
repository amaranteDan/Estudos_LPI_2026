### 🎯 Objetivo do Projeto
A ideia principal deste projeto é criar um repositório no Git com os estudos voltados para a certificação **LPI (Linux Professional Institute)** e a prática da linguagem **Shell Script**.

*   **Prática Real:** Todo o conteúdo postado aqui reflete o que foi testado integralmente na prática.
*   **Foco em Automação:** O objetivo é desenvolver uma maior interação com o Shell/Linux em ambientes de desenvolvimento e manutenção de scripts para automatização de rotinas.
